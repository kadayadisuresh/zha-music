# Phase 10: Sleep Timer & Crossfade

## Objective
Implement advanced playback features: Crossfade and Sleep Timer.

## Context
Extends `AudioEngine` and `playbackStore` to support seamless track transitions and scheduled playback termination.

## Tasks
1. **AudioEngine Refactor** (auto):

2. **Settings UI** (auto):

   - Now Playing overlay updates: Sleep timer countdown and cancel button.
3. **Sleep Timer Logic** (auto):

   - 3-second fade-out before pausing.
4. **Verification** (auto):
   - Test crossfade 5s/0s.
   - Test sleep timer set/cancel.
   - Test "End of song" mode.
