<task type="auto">
  <name>Task 2: Implement Image Proxy with Color Extraction</name>
  <files>frontend/app/api/proxy/image/route.ts</files>
  <action>
    1. Create `frontend/app/api/proxy/image/route.ts`:
       - Accepts `url` parameter (the external image URL).
       - Validates that the URL belongs to trusted domains (`*.googleusercontent.com`, `*.ytimg.com`).
       - Fetches the image as a Buffer.
       - Uses `colorthief` (node version) to get the dominant RGB color.
       - Returns the image buffer with:
         - `Content-Type: image/jpeg` (or matching source)
         - `X-Dominant-Color: #RRGGBB` header (Per D-02)
         - `Cache-Control: public, max-age=86400` (24h caching). (Per D-02)
       - Implements fallback placeholder for failures. (Per D-02)
  </action>
  <verify>
    <automated>curl -I "http://localhost:3000/api/proxy/image?url=..." | grep X-Dominant-Color</automated>
  </verify>
  <done>Proxy serves images with dominant color headers and correct caching.</done>
</task>
