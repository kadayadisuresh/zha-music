  <name>Task 1: Implement switchTracks with Micro-Crossfade</name>
  <files>frontend/lib/audio/AudioEngine.ts</files>
  <action>
    Implement a `switchTracks(automatic: boolean)` method in AudioEngine.
    If `automatic` is true (e.g., triggered by monitor loop when song ends):
    1. Start playing the `inactivePlayer` (which was pre-fetched).
    2. Over 200ms (using `requestAnimationFrame` for ramping):
       - Fade out `activePlayer.volume` from target to 0.
       - Fade in `inactivePlayer.volume` from 0 to target.
    3. After 200ms:
       - Stop `activePlayer`, set its `src` to empty.
       - Update `activeIdx` to point to the new player.
       - Update playback store with new track info.
    If `automatic` is false (manual skip):
    - Perform an instant switch: stop active, start inactive, swap index immediately without volume ramping.
    This implements "Micro-Crossfade" (GAD 2).
  </action>
